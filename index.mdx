## What is Blocklet Server?

Blocklet Server provides a complete runtime environment for running Blocklet decentralized applications. It acts as the platform layer between your infrastructure and your Blocklet apps — analogous to how iOS runs iPhone apps.

<CardGroup cols={3}>
  <Card title="Lifecycle Management" icon="arrows-spin">
    Install, start, stop, upgrade, and remove Blocklets with full state tracking and rollback support.
  </Card>
  <Card title="Smart Routing" icon="route">
    Nginx-based request routing with automatic configuration, SSL termination, and per-blocklet domain management.
  </Card>
  <Card title="DID Authentication" icon="shield-check">
    Built-in decentralized identity (DID) authentication and role-based access control (RBAC) for all Blocklets.
  </Card>
  <Card title="Zero-Downtime Deploys" icon="circle-check">
    Blue-green deployment ensures upgrades happen without dropping traffic.
  </Card>
  <Card title="Web Dashboard" icon="display">
    Visual admin interface for managing Blocklets, viewing logs, and configuring your server.
  </Card>
  <Card title="SDK Support" icon="code">
    Node.js and browser SDKs let Blocklet apps integrate with the server's auth, notifications, and data APIs.
  </Card>
</CardGroup>

## Get started in 3 steps

<Steps>
  <Step title="Install Blocklet Server">
    Pull the Docker image or install via npm:

    ```bash
    docker pull arcblock/blocklet-server
    docker run -d -p 80:80 -p 443:443 -v ~/blocklet-data:/data arcblock/blocklet-server
    ```

    Or with npm:

    ```bash
    npm install -g @blocklet/cli
    blocklet server start -a
    ```
  </Step>
  <Step title="Access the dashboard">
    Open [http://localhost](http://localhost) in your browser. Complete the initial setup wizard to configure your server domain, create your admin account, and generate your server DID.
  </Step>
  <Step title="Install your first Blocklet">
    From the dashboard, click **Store** to browse available Blocklets, or use the CLI to install directly:

    ```bash
    blocklet install https://store.blocklet.dev/blocklets/z8ia29UsENBg6tLZUKi2HABj38Cw1LmHZo8e
    ```
  </Step>
</Steps>
